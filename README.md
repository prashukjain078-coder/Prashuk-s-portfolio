# Prashuk — ML Infrastructure & Systems Engineer

Personal portfolio site with a Three.js particle-sphere splash intro.

## Structure
- `index.html` — splash/entry page (animated particle sphere hero). This is what loads first on GitHub Pages.
- `portfolio.html` — the full portfolio (tunnel scene, terminal simulation, project carousel, Model Serving Control Room dashboard). Reached via the "Enter" button on the splash page.

## Live site
https://prashukjain078-coder.github.io/Prashuk-s-portfolio/

## Stack
- GSAP 3.12.5 + ScrollTrigger
- Anime.js 3.2.2
- Three.js r128
- Lenis 1.1.14
- Vanilla HTML/CSS/JS (no build step)

## Run locally
Open `index.html` in a browser — no build or install required.
